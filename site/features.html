Features

What's in the box.

Freemium is everything below. Premium adds structured analysis on top — see pricing.

Title-to-query expansion

Paste a full research title or a natural-language question. Corpus Smith classifies the input (title, question, keywords), extracts the working subject, and expands it deterministically into a query plan that spans every source. No keyword tuning, no prompt engineering.

$ corpussmith search "effects of microplastics on marine invertebrate reproduction"
→ subject: microplastics AND marine invertebrate AND reproduction
→ variants: review-bias, recency-bias, multilingual
→ 20 sources queried in parallel

Source-type and trust labels

Every harvested record is classified along two axes: source type (journal article, preprint, book chapter, dataset, thesis, report, grey literature) and trust label (peer-reviewed, peer-or-preprint, community, unclassified). The labels ride with the record through all exports.

Project workspaces

Every project is a folder. No hidden database, no server. Settings live in project.toml, raw API payloads in sources/, downloaded PDFs in downloads/, your own imports in corpus/, and exports in exports/. Portable, diffable, version-controllable.

Provenance-aware exports

  • BibTeX — for LaTeX and Overleaf. Trust label goes into the note field.
  • CSL-JSON — for Zotero, Pandoc, and any citeproc-driven pipeline.
  • Annotated Markdown — a reading-ready bibliography with per-record trust callouts.

Knowledge dataset builder

Run corpussmith build to extract, chunk, and hash every document in your corpus into a JSONL dataset. Ideal for training retrieval indices, feeding RAG systems, or just searching your reading pile.

Multilingual support

Unicode-aware throughout. Accent-stripped matching so that Démeter finds Demeter. Domain lexicons for biology, neuroscience, psychiatry, classics, philosophy, and music. --multilingual expands queries into variant languages when it makes sense to.

Beginner onboarding wizard

First time? corpussmith new my-research launches a plain-English wizard — no jargon, one question per screen, defaults you can accept with Enter.

Zero mandatory dependencies

The freemium core uses only the Python standard library. Optional pypdf and python-docx unlock PDF and DOCX extraction; everything else is built-in.
