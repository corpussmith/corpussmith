Features

What's in the box

Freemium is everything below. Premium adds structured analysis on top — see pricing.

Core

Title-to-query expansion

Paste a full research title or natural-language question. ScholarForge classifies the input, extracts the subject, and expands it deterministically into a query plan spanning every source.

Core

Trust labels

Every harvested record is classified: source type (journal, preprint, book, dataset, thesis, report) and trust label (peer-reviewed, peer-or-preprint, community, unclassified).

Core

Project workspaces

Every project is a folder. Settings in project.toml, API payloads in sources/, PDFs in downloads/, imports in corpus/, exports in exports/. Portable and version-controllable.

Core

Provenance-aware exports

BibTeX for LaTeX/Overleaf. CSL-JSON for Zotero/Pandoc. Annotated Markdown for reading. Trust labels ride with every record in every format.

Core

Knowledge dataset builder

Run scholarforge build to extract, chunk, and hash every document into a JSONL dataset. Feed RAG systems, train retrieval indices, or search your reading pile.

Core

Multilingual support

Unicode-aware throughout. Accent-stripped matching so Démeter finds Demeter. Domain lexicons for biology, neuroscience, psychiatry, classics, philosophy, music.

Core

Beginner onboarding wizard

scholarforge new my-research launches a plain-English wizard — no jargon, one question per screen, defaults you can accept with Enter.

Core

Zero mandatory dependencies

The freemium core uses only the Python standard library. Optional pypdf and python-docx unlock PDF and DOCX extraction.
