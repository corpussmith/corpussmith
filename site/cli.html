CLI reference

Nine verbs. That's all.

Every command accepts --project DIR. If omitted, CorpusSmith walks up from the current directory to find the nearest project root.

new

Create a research project. On a TTY with no flags, launches the wizard.

corpussmith new [DIR]
    [--wizard | --skip-wizard]
    [--name NAME]
    [--kind thesis|article|book|exploration]
    [--goal "..."]
    [--field humanities|social|life|physical|cs|interdisciplinary]
    [--recency 5y|10y|any]
    [--trust-floor peer_reviewed|peer_or_preprint|any]
    [--languages en,fr,...]

search

Paste a title or question. CorpusSmith expands it and harvests every source.

corpussmith search [--project DIR] "title or question"
    [--max-results N]       # default 25 per source
    [--min-score N]         # default 10
    [--no-download]         # metadata only
    [--dry-run]             # show query plan only
    [--multilingual]        # expand into other languages
    [--recent]              # bias toward recent years
    [--from FILE]           # seed from a PDF/DOCX/MD/TXT brief
    [--no-enrich]           # skip OpenAlex concept lookup

import

Copy your own files into the project corpus.

corpussmith import [--project DIR] SOURCE_DIR

build

Extract, chunk, and hash every document into a JSONL dataset.

corpussmith build [--project DIR]
    [--chunk-size N]        # default 3500 chars
    [--overlap N]           # default 350 chars
    [--source corpus|downloads|all]

export

Provenance-aware bibliography in three formats.

corpussmith export [--project DIR] --format bibtex|csljson|markdown
    [--output PATH]
    [--title "..."]         # Markdown only

review-project

Terminal report of the current project state.

corpussmith review-project [--project DIR]

cache

Inspect, probe, and clear the local concept cache.

corpussmith cache stats                  # record count and health
corpussmith cache clear [--yes]          # delete all cached records
corpussmith cache show "probe title"     # find nearest cached match
corpussmith cache export PATH            # dump as JSONL or CSV

config

Inspect or initialise the global config.

corpussmith config                # show current key status
corpussmith config --init         # write a commented template

premium v3.4+

Premium activation status and feature listing.

corpussmith premium               # show status
corpussmith premium --features    # list available seams

Legacy verbs

The pre-3.4 verbs harvest, forge, and pipeline remain available for backwards compatibility.

Global flags

--version       show version and exit
--no-banner     suppress the startup banner
--no-color      disable ANSI colour output
--verbose       verbose logging
-h, --help      show help
