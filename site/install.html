Install

One minute, any platform

Python 3.8 or newer. Git. That is the whole prerequisite list.

Standard install

git clone https://github.com/corpussmith/corpussmith.git && cd corpussmith && pip install -e . 
# verify
corpussmith --version

Windows, first-time user

A step-by-step walkthrough for users with no Python background:

Beta Tester Guide →

First project

mkdir my-research
corpussmith new my-research
# wizard guides you through seven plain-English questions

corpussmith search --project my-research "your title or question"
corpussmith export --project my-research --format markdown

API keys optional

# Generate config template
corpussmith config --init

# Edit ~/.corpussmith.toml with your keys
corpussmith config        # verify key status

Upgrading

cd corpussmith
git pull
pip install -e .

Uninstall

pip uninstall corpussmith

Your project folders are plain files — delete them if you want to.
