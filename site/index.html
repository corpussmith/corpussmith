Install in one minute →

What it does

Corpus Smith is built for researchers, students, and authors who want calm tools. You give it a research goal in plain language. It builds the query plan, harvests the sources, keeps the provenance, and gets out of the way.

$ corpussmith new my-thesis
$ corpussmith search --project my-thesis "neural correlates of OCD compulsions"
$ corpussmith export --project my-thesis --format bibtex

Why it exists

Generic search engines throw away the scholarly signal. AI agents hallucinate citations. Reference managers capture what you already found, not what is out there. Corpus Smith sits before all of them: a calm harvester that surfaces what the open-access ecosystem actually has, labelled so you can trust it.

Three promises

Deterministic

The freemium core has no model in the loop. Same input, same query plan, same record set — reproducible across machines and years.

Provenance-first

Every record carries a source type and trust label — peer-reviewed, preprint, grey literature, book, dataset. The label is in the BibTeX note, the CSL-JSON field, and the annotated Markdown.

Scholar-first

Humanities-friendly. Multilingual-friendly. Unicode and accent-aware. Domain lexicons for biology, neuroscience, psychiatry, classics, philosophy, and music. Zero mandatory third-party packages.

Who it is for

  • PhD candidates assembling literature review matrices
  • Authors writing long-form books with hundreds of sources
  • Independent researchers without institutional database access
  • Clinicians building evidence files that have to survive audit
  • Anyone who would rather read than fight their tooling
